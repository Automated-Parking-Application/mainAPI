Prerequisite of Spring Boot:
  JDK
  Maven Apache
  Maven Wrapper

Database:
  MySQL (with name of database: apa)

  spring.datasource.username=root
  spring.datasource.password=IamLeo0708

Tools:
  MySQL Workbench or TablePlus

Install:
./mvnw clean install

Run:
./mvnw spring-boot:run
